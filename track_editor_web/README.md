# 赛道编辑器 (Track Editor)

这是一个基于高德地图的Web赛道编辑工具，用于创建和编辑赛车游戏的赛道数据。

## 功能特性

- ✅ 可视化绘制起点多边形电子围栏
- ✅ 可视化绘制终点多边形电子围栏
- ✅ 添加多个检查点（多边形电子围栏）
- ✅ 为检查点设置规则（速度限制、最少点数、时间限制等）
- ✅ 导出为标准JSON格式
- ✅ 直接下载JSON文件到本地

## 使用前准备

### 1. 获取高德地图API Key

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册/登录账号
3. 创建应用并获取 Web端 JS API 的 Key

### 2. 配置API Key

打开 `index.html` 文件，找到第252行：

```html
<script src="https://webapi.amap.com/maps?v=2.0&key=YOUR_AMAP_KEY"></script>
```

将 `YOUR_AMAP_KEY` 替换为你自己的高德地图API Key。

## 使用方法

### 启动编辑器

直接在浏览器中打开 `index.html` 文件即可使用。

### 创建赛道

1. **填写赛道基本信息**
   - 输入赛道名称
   - 填写赛道简介
   - 设置赛道长度（米）

2. **绘制起点围栏**
   - 点击"起点围栏"按钮
   - 在地图上点击添加多边形顶点（至少3个点）
   - 双击完成绘制

3. **绘制终点围栏**
   - 点击"终点围栏"按钮
   - 在地图上点击添加多边形顶点
   - 双击完成绘制

4. **添加检查点**
   - 点击"检查点"按钮
   - 在地图上绘制检查点区域
   - 输入检查点名称
   - 可以添加多个检查点

5. **设置检查点规则**（可选）
   - 选择要设置规则的检查点
   - 选择规则类型：
     - **最少点数规则**: 要求在检查点内至少有N个点的速度低于某阈值
     - **速度限制**: 要求速度在指定范围内
     - **时间限制**: 要求在限定时间内通过
   - 填写规则参数和描述
   - 点击"添加规则"

6. **导出赛道**
   - 点击"生成JSON"按钮查看JSON数据
   - 点击"下载JSON文件"保存到本地

### 导入到App

1. 将下载的JSON文件复制到项目的 `assets/tracks/` 目录
2. 更新 `assets/tracks/tracks_index.json` 文件，添加新文件名到列表中
3. 重新编译运行Flutter应用

## JSON格式说明

导出的JSON文件结构如下：

```json
{
  "name": "赛道名称",
  "description": "赛道描述",
  "length": 3500.0,
  "startPolygon": [
    {"latitude": 40.0079, "longitude": 116.5119},
    {"latitude": 40.0080, "longitude": 116.5119},
    ...
  ],
  "endPolygon": [
    {"latitude": 40.0085, "longitude": 116.5125},
    ...
  ],
  "checkPoints": [
    {
      "name": "检查点1",
      "sequence": 1,
      "polygon": [...],
      "description": "检查点描述",
      "rules": [
        {
          "ruleType": "min_points",
          "parameters": {
            "min_points": 2,
            "speed_threshold": 40.0
          },
          "description": "规则描述"
        }
      ]
    }
  ]
}
```

## 注意事项

- 多边形至少需要3个顶点
- 建议按顺时针或逆时针顺序绘制多边形
- 检查点应按赛道顺序依次添加
- 确保起点和终点不重叠
- 测试赛道时建议先在模拟器或小范围测试

## 技术栈

- HTML5 + CSS3
- JavaScript (原生)
- 高德地图 JS API v2.0

## 常见问题

**Q: 地图无法加载？**
A: 检查API Key是否正确配置，确保网络连接正常。

**Q: 绘制的多边形不准确？**
A: 可以清除后重新绘制，或者放大地图以获得更精确的定位。

**Q: 如何修改已绘制的多边形？**
A: 当前版本不支持编辑，需要删除后重新绘制。
